# 「入門30代」 寄稿原稿

__タイトル: 遠回りでも接続はできる：30歳からモバイル開発を始めた私が今考えていること__

---

## 概要

30歳でモバイル開発の世界に飛び込んだエンジニアが、遠回りしてきたからこそ気づけたことを言葉にした原稿です。

経験ゼロの状態から始まった不格好なスタート、それまでの別領域での仕事が思わぬ場面で活きてきた経験、若手と同じ走り方をしなくていいと気づいたこと、35歳以降に感じる役割の変化——そうした過程を通じて、「遠回りは遅れではなく接続である」という考え方に辿り着くまでの軌跡を綴っています。

成功談でも自己啓発的なキャリア論でもなく、地味で不格好なリアルを正直に書いた一人のエンジニアの記録です。今まさに30代から何かを始めようとしている方、あるいは自分の遠回りをどう意味づけるか迷っている方に、何かが届けば幸いです。
